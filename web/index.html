Gladiator Arena

Retro PvP fighting game — Web3 integrated NFT minting

Download

Download the latest Windows build (x64). If you want in-browser play, see the Project docs — WebAssembly build is optional.

Download Windows (x64)

Quick install

  1. Download the ZIP for Windows.
  2. Extract the ZIP and run PvP_BlockChain.exe (or the provided exe).
  3. Open the Main Menu and enter two wallet addresses (use Ganache for testing) to enable on-win NFT minting.

Live bridge server

If you want to test the Web3 integration locally, start the Node bridge on http://localhost:3000 as documented in the repo, then run the game.

Developer & CI

The project includes a GitHub Actions workflow that can build Windows artifacts and upload them as workflow artifacts. See the /.github/workflows directory.

Hosting this page

You can host this folder using GitHub Pages. See /docs/hosting.md for step-by-step instructions.

If you don't want to host the ZIP in the repo, upload the build ZIP to GitHub Releases and either:

  • Replace the button href with the direct release asset URL (from GitHub Releases).
  • Or copy the ZIP into web/releases/GladiatorArena-latest.zip in this repository and redeploy GitHub Pages.
